# Expense Tracker React App

A simple expense tracker built with React.

## Features

- Add expenses
- Delete expenses
- Category breakdown
- Total expense calculation
- Currency conversion using Frankfurter API
- Responsive design

## Tech Stack

React  
JavaScript  
CSS  
Frankfurter API
